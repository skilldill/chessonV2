# Chesson Admin Panel

Локальная админ-панель для получения защищенной статистики:
- количество игр
- количество игр без результата
- количество игр с результатом
- количество зарегистрированных пользователей
- управление пользователями (поиск, блокировка, разблокировка, повторная отправка email верификации, редактирование, удаление)

## Запуск

1. Перейдите в папку `admin`
2. Установите зависимости: `npm install`
3. При необходимости скопируйте `.env.example` в `.env`
4. Запустите: `npm run dev`

Панель отправляет header `x-admin-secret` со значением из `VITE_ADMIN_API_KEY`.
По умолчанию используется `local-chesson-admin-secret`.

По умолчанию панель ходит напрямую в `http://localhost:4000/api` через `VITE_ADMIN_API_BASE_URL`.
При необходимости можно использовать прокси Vite через `VITE_API_PROXY_TARGET`.
